## Proyecto Final – Backend - Aplicación Web Full Stack con Angular y Spring Boot

### Descripción del Proyecto
Este repositorio contiene el desarrollo del backend del proyecto final académico, implementado con Spring Boot. 
La aplicación forma parte de un sistema web full stack para la gestión de usuarios e inventario de la empresa simulada **Ishedar Petroleum**.
El backend expone una API REST protegida mediante autenticación JWT y control de acceso basado en roles.

## Tecnologías Utilizadas
- Spring Boot.
- Spring Security.
- JWT (JSON Web Token).
- Spring Data JPA.
- MySQL.
- Maven.
